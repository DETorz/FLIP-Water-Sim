# Fluid Simulator
- This project is a 2D fluid simulator visualized in React. The source code that was used for this project can be found [here](https://github.com/SebLague/Fluid-Sim/tree/Episode-01). 

## React structure
- `flip-sim/` - Application
- 'src/FLIP/` - Simulation components

## Setup & running
```bash
cd flip-sim
npm install
npm start 
```
or 
```bash
npm install -g serve
serve -s build
```
for the optimized build

## How to use
- You can change the gravity of the simulation by holding left click and releasing in the desired direction.
